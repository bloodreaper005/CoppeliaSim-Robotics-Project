# Notice

This repository is a cleaned, GitHub-ready version of a CoppeliaSim mobile-manipulator project.

The original shared PDF report was intentionally omitted because it contained other students' names and email addresses. Add your own report, screenshots, and results before publishing the repository as your project portfolio.

Several rigid-body motion, kinematics, and trajectory helper functions follow the notation and structure of the Modern Robotics textbook/course library:

- `Adjoint.m`
- `AxisAng3.m`
- `CartesianTrajectory.m`
- `CubicTimeScaling.m`
- `FKinBody.m`
- `JacobianBody.m`
- `MatrixExp3.m`
- `MatrixExp6.m`
- `MatrixLog3.m`
- `MatrixLog6.m`
- `NearZero.m`
- `QuinticTimeScaling.m`
- `ScrewTrajectory.m`
- `TransInv.m`
- `TransToRp.m`
- `VecTose3.m`
- `VecToso3.m`
- `se3ToVec.m`
- `so3ToVec.m`

Project-specific files include the controller, trajectory pipeline, state update, CSV generation, and youBot-specific pose/Jacobian utilities. Before choosing a public license for this repository, verify any course, assignment, and reference-library licensing requirements.
